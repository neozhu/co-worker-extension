# My Co-worker Extension

A Chrome/Edge extension for searching co-worker information with short name, global ID and email address.

## Features

- Search employees by short name, global ID, or email address
- Quick access through browser extension popup
- Secure connection to employee search service

## Installation

1. Download the extension from Chrome Web Store or Edge Add-ons
2. Click "Add to Chrome/Edge" to install
3. Pin the extension to your toolbar for easy access

## Usage

1. Click the extension icon in your browser toolbar
2. Enter employee information (short name, global ID, or email)
3. View search results

## Privacy

This extension:
- Does not collect any personal user data
- Does not sync data to remote servers
- Does not monitor user activity
- Only connects to authorized employee search services
